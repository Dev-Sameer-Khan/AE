# Ameera Enterprises

Ameera Enterprises is a platform for supplying genuine and quality aftermarket parts for heavy equipment, with a special focus on the India-Gulf-Turkey trade corridor. We deal in a variety of brands including Deutz, Atlas Copco, Bobcat, Cummins, and Perkins.

## 🌟 Features

- Multilingual (English & Arabic) support with dynamic language switching
- Product catalog and inquiry system
- Animated, modern UI built with React & Tailwind CSS
- Responsive layout for all devices
- FAQ section and Contact form
- Useful and accessible navigation, including mobile menu
- International shipping and logistics

## 🚀 Getting Started

**1. Clone the repository**

```bash
git clone https://github.com/YOUR-USERNAME/ameera-enterprises.git
cd ameera-enterprises
```

**2. Install dependencies**

```bash
npm install
```

**3. Start the development server**

```bash
npm run dev
```

Visit [http://localhost:5173](http://localhost:5173) in your browser.

## 🛠️ Project Structure

```
src/
  components/    // React components (Layout, ContactSection, FAQ, etc.)
  context/       // Context providers (I18nContext for languages)
  pages/         // App pages (Home, Products, About, Contact)
  assets/        // Static assets (logo, images)
```
